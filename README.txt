Cara Deploy ke Render:

1. Login ke https://render.com
2. Klik "New Web Service"
3. Hubungkan ke GitHub repo ATAU Upload file ZIP ini ke repo baru
4. Pastikan:
   - Runtime: Python 3
   - Build Command: pip install -r requirements.txt
   - Start Command: python app.py
5. Setelah deploy, buka URL publik untuk akses website
